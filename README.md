# Simple Digital Locker (C++ CLI Application)

## Overview
A lightweight, console-based Digital Locker built using C++ that allows users to add, view, and delete personal secret items securely.  
It showcases C++ concepts like Classes, Objects, Vectors, Functions, and CLI interaction, making it an ideal beginner-friendly real-world inspired project.

## Features
- Add secret items (like passwords, notes)
- View all locker items
- Delete locker items securely
- Simple and clean menu-driven interface

## Technologies Used
- C++ Programming Language
- Object-Oriented Programming (OOP)
- Vectors (`std::vector`)
- CLI Menu System
- Dynamic Data Management

## How to Run
1. Ensure `g++` compiler is installed (`g++ --version` to check).
2. Compile the program:
   ```bash
   g++ DigitalLocker.cpp -o DigitalLocker
   ./DigitalLocker
